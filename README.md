# Тестовое задание
## Разработать сервис управления подписками c двумя типами (Basic и PRO) и кэш-сервис к нему.

# Установка
## Предварительные требования:
- Java 21
- Maven 3.6
- Docker
- Docker compose
## Действия
1. Клонирование репозитория:

        git clone https://github.com/ArthurKH01/Subscription_Service.git

2.  Перейдите в каталог проекта:

        сd Subscription_Service

3. Соберите проект с помощью Maven:

        mvn clean install

4.Запустите приложение с помощью Docker Compose:

        docker-compose up
